# 🔁 第5章：無限ループからの脱出 (The Loop Breaker)
> *"狂気とは、同じコードを何度も実行しながら、違う出力を期待することである。"*

## 5.1 苦しみの無限ループ (The Infinite Loop)
なぜいつも同じようなダメな人を好きになるのか？なぜ同じパターンで失敗するのか？
それは **「無限ループ (Infinite Loop)」** のバグです。
1. 寂しい (Input) -> 2. 誰かにすがる (Process) -> 3. 一時の快楽 (Output) -> 4. 破局 (Crash) -> 5. 再起動 (Reboot to 1)

スクリプト（脚本）を書き換えない限り、キャスト（恋人）を変えてもエンディングは同じです。

## 5.2 観察者パッチ (The "Observer" Patch)
このループから抜けるには、**「観察者 (Observer)」** モードを起動するしかありません。
* **Player Mode:** 「辛い！悲しい！」（ドラマに没入）
* **Observer Mode:** 「ほう、今『悲しみ』というプロセスが走っているな」（客席から見る）

## 5.3 3つのアップデート (The 3 Major Updates)
未来を書き換えるための必須パッチ（三学）です：

1.  **Giver Protocol (布施):** 「愛されたい」から「愛する」へ。
2.  **Firewall Protocol (持戒):** 自分と他人を傷つけないためのセキュリティ設定。
3.  **Deep Learning (智慧):** 失敗を「不運」ではなく「学習データ」として処理する。痛みを教訓に変えたとき、その痛みは消える。

この3つをインストールしてコンパイルすれば、もう古いカルマ（バグ）は再現しません。
